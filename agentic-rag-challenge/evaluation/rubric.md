# Evaluation Rubric

This rubric is adapted from the planning documents that guided this repository.

## Per-response scoring dimensions

- Correctness of decision
- Completeness of evidence
- Citation quality
- Safe abstention when information is not in the catalog
- Response format compliance

## Category targets

- Citation coverage: `>95%`
- Eligibility accuracy: `>90%`
- Chain reasoning accuracy: `>80%`
- Program requirement accuracy: `>85%`
- Abstention accuracy: `100%`
- Format compliance: `>95%`

## Weighted score

```text
Overall Score =
  (Citation Coverage * 0.20) +
  (Eligibility Accuracy * 0.25) +
  (Chain Accuracy * 0.20) +
  (Program Accuracy * 0.15) +
  (Abstention Accuracy * 0.10) +
  (Format Compliance * 0.10)
```

## Interpretation

- `>85`: strong submission
- `>90`: excellent
- `100`: only if every category and formatting requirement is fully satisfied
